Hello! Welcome to my portfolio. If you are looking for more information about any of the projects below, please don't hesitate to contact me at annabel@students.olin.edu.

Website Design


In a software design class, I worked with two other students to write a program in Python to take in a user's ingredients and time constraints and search the Yummly API for recipes that met those specifications. Ingredients and their corresponding recipes were saved using MongoDB, and the application was hosted on Heroku. Additionally, users were able to sign in with their names and keep the same "pantry" ingredients between uses of the app. I worked specifically on implementing the front-end of the the project using HTML5/CSS and javascript as well as the Python back-end and API.

As part of a research team on accessible and adaptive design, I created a website for the Accessible Icon Project to promote awareness for the movement. I created the new branding for the site in Illustrator and customized the site using Wordpress. This work was done under Sara Hendren.

For a small yoga studio, I designed a website in Rapid Weaver so that students would be able to access class schedules, session fees, and more information about the studio. I also set up a newsletter subscription service for the studio and designed new branding for the studio.

For a Senator Hareiette Chandler's 2014 campaign, I designed a website, campaign material, and a new logo. Using Rapid Weaver, I set up a donation method, email subscription, and a volunteer form. (In case you were curious, she won the campaign.)

For more information about my engineering projects and other design work, please email me at annabel@students.olin.edu.
