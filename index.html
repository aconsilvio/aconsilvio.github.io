Website Design


During a team project where I made gloves that translate American Sign Language into text, I created this website to document the process and final project. Since I only had a few weeks to complete it, it is a static website made with primarily HTML/CSS. For more about my experience with the ASL gloves, please scroll down.

As part of a research team on accessible and adaptive design, I created a website for the Accessible Icon Project to promote awareness for the movement. I created the new branding for the site in Illustrator and customized the site using Wordpress. This work was done under Sara Hendren.

For a small yoga studio, I designed a website in Rapid Weaver so that students would be able to access class schedules, session fees, and more information about the studio. I also set up a newsletter subscription service for the studio and designed new branding for the studio.

I worked with two other students to write a program in Python to take in a user's ingredients and time constraints and search the Yummly API for recipes that met those specifications. We used MongoDB for our database, and the application was hosted on Heroku. I worked on the front-end development using HTML5/CSS and Javascript and the Python back-end.

For a Senator Hareiette Chandler's 2014 campaign, I designed a website, campaign material, and a new logo. Using Rapid Weaver, I set up a donation method, email subscription, and a volunteer form. (In case you were curious, she won the campaign.)

Engineering


American Sign Language Interpreting Gloves - Fall 2015. In a class at Olin, I am working with three teammates to design and develop a glove that interprets ASL fingerspelling and translates it into text on an LCD screen. Our aim to is create better interactions between those who sign and those who can't. My focus is on the software component, which uses an Arduino to process incoming data and compare it to a database of calibrated values.

Vinny the Velociraptor - Fall 2014. With four teammates, I developed an educational children's toy based off of the counter-balance motion present in velociraptor movement. This project was tested by quite a few 4th graders throughout development and after completion.

"Wait, can I try that again?" - A satisfied 4th grader

Line Following Robot - Fall 2015. Developed the software for a small robot that followed a drawn out line using IR sensors with two teammates. I focused primarily on the Arduino code and control algorithm.

Project Liberty - MakeMIT 2015. With three teammates, I helped to design and develop a bracelet that communicated via bluetooth with an Android app. The purpose of the bracelet was for college students to be able to discreetly send a text message or make an emergency call if they felt in danger.
